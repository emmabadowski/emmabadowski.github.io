Search tips
  • Multiple search terms:You can search by any number of terms at once. To enter two search terms, you can either perform two separate searches, or put a space between terms.
  • OR/AND/NOT searches: These are applied to each term you provide, and determine whether the term is required:
    • OR [default] At least one term you provided must match to return a given result.
    • AND A result must contain ALL of your AND terms to be returned.
    • NOT A result must not contain any of your NOT terms to be returned.
  • Text Searches: When searching text fields, you can use the following features:
    • wildcard searches: Add * to the beginning or end of a string to get results that start or end with the given letters. Use wildcards in the middle of a string to replace unknown letters.
    • fuzzy searches: Add ~n to the end of a string to match similar items. The larger the n the wider the range of results.
  • Numeric searches: When searching specifically in numeric fields, you can use a number of operators to search ranges more easily, such as:
    • >n: Returns all values greater than n.
    • >=n: Returns all values greater than or equal to n.
    • <n: Returns all values less than n.
    • <=n: Returns all values less than or equal to n.
    • n1-n2: Returns n1, n2, and all values in-between.
    • n1 n2: Putting a space between two or more numbers will perform an OR search on all numbers. This can be combined with the other operators as well!
